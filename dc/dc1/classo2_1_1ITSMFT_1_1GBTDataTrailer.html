enum o2::ITSMFT::GBTDataTrailer::PacketStates : int

Definition of ITS/MFT GBT trailer: 80 bits long word In CRU data it must be the last word of the payload

bits 0 : 27, Lanes stops received bits 28 : 31, not used bits 32 : 59, Lane timeouts received bits 60 : 63, not used bits 64 : 71, State of GBT_Packet: 4: lane_timeouts, if at least 1 lane timed out 3: lane_starts_violation, if at least 1 lane had a start violation 2: packet_overflow, if max number of packets reached 1: transmission_timeout, if timeout of transmission (lanes) 0: packet_done, if Packet finished
