Detailed Description

Helper class to format ALICE HLT data blocks for transport in messaging system.

Data blocks in the ALICE HLT are described by a block descriptor (data type, size, specification) and the actual data somewhere in memory. For transporting them in a message, block descriptors and payloads are written as a sequence, every block descriptor directly followed by its payload
