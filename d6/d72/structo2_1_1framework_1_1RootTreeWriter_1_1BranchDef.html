Detailed Description

template<typename T, typename KeyType = key_type, typename KeyExtractor = DefaultKeyExtractor>
struct o2::framework::RootTreeWriter::BranchDef< T, KeyType, KeyExtractor >

BranchDef is used to define the mapping between inputs and branches. It is always bound to data type of object to be written to tree branch. can be used with the default key type and extractor of the RootTreeWriter class, or by using a custom key type and extractor

The same definition can handle more than one branch as target for writing the objects Two callbacks, getIndex and getName have to be provided together with the number of branches
