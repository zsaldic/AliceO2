Detailed Description

template<typename BASE, typename T>
class gNeric::rc_mixin< BASE, T >

Mixin component is used with different data types.

the mixin class Each mixin component has a member of the specified type. The container level exports the following data types to the outside:

  • wrapped_type the data type at this level
  • mixin_type composed type at this level
  • types mpl sequence containing all level types
  • level a data type containing the level
